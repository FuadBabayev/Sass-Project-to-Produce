Syntactically Awesome Style Sheets

Sass provides many built-in modules which contain useful functions. These modules can be loaded with the @use rule like any user-defined stylesheet. All built-in module URLs begin with sass: to indicate that they’re part of Sass itself. Before the Sass module system was introduced, all Sass functions were globally available at all times. Many functions still have global aliases.
